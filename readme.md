### Prerequisites

1. Create a free atlas account => https://www.mongodb.com/cloud/atlas
2. Install [now](https://zeit.co/) globally
   > `yarn global add now`
   > or
   > `npm install now -g`

### Install

`$ yarn install`

### Run

`now dev`

### Visit

http://localhost:3000/api/graphql
https://serverless-graphql-api-example.itsmylife.now.sh/api/graphql
